# 如果你算力不准：
①第一检查挖矿软件配置及内核配置，是否设置超过多少分钟没有成功提交重启内核

②查看你服务器的硬件配置及软件带宽，配置过低可能导致转发性能不足，导致重发及超时

③检查你服务器的网络是否占用超过60%以上，是的话加带宽
